# IMAI.Studio Shopify App Support

Last updated: April 18, 2026

## Support Contact

If you need help with the IMAI.Studio Shopify app, contact us at:

- Email: tech@imai.studio
- Website: https://www.imai.studio

## What we can help with

We can help with:

- app installation issues
- Shopify authentication or embedded app loading issues
- connecting or removing your IMAI.Studio API key
- billing and subscription issues
- credit synchronization issues
- generation failures
- library access problems
- asset import issues
- webhook and setup troubleshooting

## Information to include in your support request

To help us investigate quickly, include:

- your Shopify store domain
- the page where the issue happened
- the approximate time of the issue
- screenshots or screen recordings if available
- any visible error message
- whether the issue affects generation, billing, connection, or import

Do not send passwords or private payment information by email.

## Typical setup flow

1. Install the app from Shopify.
2. Open the app inside Shopify Admin.
3. Go to Settings.
4. Generate an IMAI.Studio API key in IMAI.Studio.
5. Paste the key into the app and save it.
6. Start using the app's design, marketing, product, library, and billing features.

## Billing support

The app uses Shopify's billing system for paid app subscriptions.

If you are charged through Shopify for an app subscription and believe there is an error, contact us at `tech@imai.studio` with:

- your store domain
- the plan name
- the billing date
- a screenshot of the relevant Shopify billing screen if possible

For information about data handling and privacy requests, see the app Privacy Policy page.

## Availability

We aim to respond to support requests as quickly as possible during normal business operations. Response times can vary based on issue severity and request volume.
